# Node.js Simple Server

This is a simple server project built with **Node.js** using the built-in **http** and **fs** modules.  
The project was created to handle various routes and perform specific tasks based on incoming requests**.

## Assignment Requirements

This project covers the following requirements:

- Create a server using the `http` module
- Run the server on port `5500`
- Handle multiple routes
- Send different text responses for different URLs
- Create a file using `fs.writeFile()`
- End every response using `res.end()`

## Features

- Home route support
- About route support
- Contact route support
- File write route support
- File creation using Node.js file system module

## Technologies Used

- Node.js
- HTTP Module
- File System (`fs`) Module
